Supported VPN Protocols

  • VLESS: Modern protocol with UUID authentication
  • VMESS: Base64 encoded configuration
  • Trojan: Password-based authentication
  • Shadowsocks: Encrypted proxy protocol
  • ShadowsocksR: Enhanced version with obfuscation
  • Hysteria: High-performance protocol
  • Hysteria2: Latest version with improvements
  • TUIC: QUIC-based protocol

How to Use

  1. Paste VPN links in the input section
  2. Click "Parse Links" to extract configurations
  3. Test accounts to check connectivity
  4. Generate final configuration from successful tests
  5. Download or upload to GitHub
